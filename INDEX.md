## 🆘 TROUBLESHOOTING RAPIDE

### L'app ne démarre pas?

```bash
pip install customtkinter
python vinted_app/main.py
```

→ Voir `GUIDE_LANCEMENT.md` pour plus

### Erreurs d'import?

```bash
cd vinted_app
python main.py
```

→ Consultez DEV_GUIDE.md

### DB corrompue?

1. Supprimer: `vinted_app/data/vinted.db`
2. Relancer l'app
3. Tout se régénère automatiquement

### J'ai d'autres questions?

→ Lire: `README.md` (FAQ complète)
